Libraries

Libraries are the backbone of R Studio. You must install new libraries before using them on every computer. It only takes a few seconds for most libraries.

Install a Library

You will need to install new libraries the first time you use them.

install.packages(“library_name_goes_here”)

Use a Library

library(“library_name_goes_here”)

Get library help

If you don’t know what a library does or
